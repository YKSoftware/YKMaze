# 概要
YKMaze はひたすらダンジョンを探検し続ける暇つぶしに最適なレトロゲームです。
体力のある限り、ひたすらゴールを目指すだけの迷路ゲームアプリ。落ちているお肉に拾ったカレー粉をかけて食べれば 3 倍の回復力！あなたはどこまで行けるかな…。
